# Documentación de RCNG

Esta carpeta reúne la documentación técnica y funcional del proyecto.

## Alcance

RCNG se distribuye en dos bloques:

- `Launcher_RNC/`, encargado del despliegue y arranque;
- `NRC_APP/`, que contiene la aplicación, sus datos y sus herramientas.

## Resumen

RCNG ofrece una consola unificada para:

- consulta remota de equipos Windows;
- acceso por RDP, VNC, CMD remota, PowerShell remota y Explorer++;
- ejecución de scripts integrados y personalizados;
- mantenimiento de inventario local de equipos y extensiones;
- configuración de entorno;
- uso de utilidades auxiliares como Chat, ExtFinder y WOL.
