# NRC_APP

`NRC_APP` es la carpeta operativa principal de RCNG. Contiene la aplicación, sus módulos, recursos, catálogos y documentación.

## Contenido principal

- [`LazyWinAdmin_v8.0.ps1`](LazyWinAdmin_v8.0.ps1): interfaz WinForms principal.
- [`sections/`](sections/): menús y áreas funcionales.
- [`modules/`](modules/): lógica compartida.
- [`database/`](database/): persistencia local y configuración JSON.
- [`app/`](app/): subaplicaciones internas.
- [`scripts/`](scripts/): scripts auxiliares e integrados.
- [`tools/`](tools/), [`libs/`](libs/), [`vnc/`](vnc/), [`icos/`](icos/): recursos y herramientas del proyecto.
- [`doc/`](doc/): documentación técnica y funcional.

## Documentación

El índice de documentación está disponible en [`doc/README.md`](doc/README.md).
